# 更新说明

## v0.1.7

- 中文游戏名搜索改为自动生成英文搜索候选，不再只提示用户去配置别名。
- 内置 `史诗迷你游戏 -> Epic Minigames` 等常见中文游戏名，并保留用户自定义别名作为兜底。
- 新增中文关键词规则翻译和可选在线翻译兜底，提高中文输入搜到正确 Roblox 游戏的概率。
- 多个搜索候选会合并后统一按真实游戏名匹配度排序，避免 Roblox 搜索接口返回不相关的第一条结果。

## v0.1.6

- 修复 AstrBot v4.25.5 不支持 `dict` 配置类型导致插件无法加载的问题。
- `game_aliases` 配置类型改为 AstrBot 支持的 `object`。

## v0.1.5

- 修复同一条 `/游戏搜索` 消息可能被多个命令处理器重复回复的问题。
- 新增游戏名别名映射，支持把中文名或简称转换成 Roblox 英文游戏名。
- 内置部分常见中文别名和 `SCR -> Stepford County Railway`。
- 游戏名匹配增加首字母缩写匹配，改善短缩写搜索体验。
- 中文名没有匹配结果时，提示用户尝试英文名或在配置里添加别名映射。

## v0.1.4

- 新增 `/游戏搜索` 作为 `/roblox游戏搜索` 的短别名。
- 修复使用 `/游戏搜索 游戏名` 时命令名没有被正确剥离，导致搜索词带上命令名后返回不相关游戏的问题。
- 游戏名搜索增加最低匹配分配置，候选结果和输入名差距太大时不再强行返回。

## v0.1.3

- 新增 `/游戏ID搜索` 作为 `/roblox游戏ID搜索` 的短别名。
- 修复使用 `/游戏ID搜索 277751860` 时命令名没有被正确剥离，导致数字 ID 被误判为游戏名的问题。

## v0.1.2

- 修复游戏名搜索偶尔返回错误游戏的问题。
- 游戏名搜索现在会收集多条 Roblox 搜索结果，并按名称匹配度、赞助状态、在线人数、好评数和原始排序进行稳定选择。
- 年龄组补充查询改为按 `universeId` 精确匹配，避免使用同名搜索时拿到其他游戏的年龄组。

## v0.1.1

- 新增 `/roblox游戏ID搜索` 指令，将游戏名搜索和 ID 搜索分开。
- 增加 Roblox API 请求节奏控制，避免短时间请求过多触发 429。
- 增加 429 退避重试配置。
- 服务器扫描默认收紧为更保守的分页和页数。
- 服务器接口被限流时返回部分统计结果，不再直接让整条查询失败。

## v0.1.0

- 初版 Roblox 游戏搜索。
- 支持文本和 HTML 生图输出。
- 支持游戏详情、年龄组、好评度、在线人数和服务器状态展示。
