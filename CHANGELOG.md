# 更新说明

## v0.1.2

- 修复游戏名搜索偶尔返回错误游戏的问题。
- 游戏名搜索现在会收集多条 Roblox 搜索结果，并按名称匹配度、赞助状态、在线人数、好评数和原始排序进行稳定选择。
- 年龄组补充查询改为按 `universeId` 精确匹配，避免使用同名搜索时拿到其他游戏的年龄组。

## v0.1.1

- 新增 `/roblox游戏ID搜索` 指令，将游戏名搜索和 ID 搜索分开。
- 增加 Roblox API 请求节奏控制，避免短时间请求过多触发 429。
- 增加 429 退避重试配置。
- 服务器扫描默认收紧为更保守的分页和页数。
- 服务器接口被限流时返回部分统计结果，不再直接让整条查询失败。

## v0.1.0

- 初版 Roblox 游戏搜索。
- 支持文本和 HTML 生图输出。
- 支持游戏详情、年龄组、好评度、在线人数和服务器状态展示。
